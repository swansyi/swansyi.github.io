2.18

installation of iSALE

install dependency

follo the wiki: https://github.com/isale-code/isale-wiki/wiki/Installation-instructions

  • install qt3 (make, make install)
  • install iSALE with ViMOD(make, make install)

Run models using iSALE

  • create two filefold “work” and “output”. One for init file, another for output to store

  • open terminal in corresponding fold, then

    ./iSALE2D
    or

    ./iSALE2D &

to run in background

  • output file can be seen at “output.txt0”
