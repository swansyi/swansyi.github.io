Installation of Ubuntu and Enviroment Setting

Installation of Ubuntu

just follow the steps of installation.
Then reboot the device.
Then from the settings to change the language to show.

install Anaconda2

  • get the files from web
    $wget http://link of Anaconda2
  • installation
    $bash Anaconda2.sh
  • activate conda’s bash envrioment in current shell session:
    eval “$(/home/xgh/anaconda2/bin/conda shell.YOUR_SHELL_NAME hook)”

    add psplib to python path

    vi .bashrc
    open the bashrc
    add the path to bashrc export PYTHONPATH=/YOUR_PATH:$PYTHONPATH
    or export PATH=<你要加入的路径1>:<你要加入的路径2>: …… :$PATH
    last $source .bashrc

installation of iSALE

install dependency

follo the wiki: https://github.com/isale-code/isale-wiki/wiki/Installation-instructions

  • install qt3 (make, make install)
  • install iSALE with ViMOD(make, make install)

Run models using iSALE

  • create two filefold “work” and “output”. One for init file, another for output to store

  • open terminal in corresponding fold, then

    ./iSALE2D
    or

    ./iSALE2D &

to run in background

  • output file can be seen at “output.txt0”
